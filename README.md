# 井口 8192
井口 8192 (Iguchi 8192) is a clone of [Gabriele Cirulli's 2048](http://gabrielecirulli.github.io/2048/), using animated gifs of Iguchi Yuka and other voice actresses for tiles. 

### Contributions

 - [Gabriele Cirulli](https://github.com/gabrielecirulli) created [2048](http://gabrielecirulli.github.io/2048/) which this is based on.
